# Smart_Ring
Данный проект был создан в рамках учебной программы СОШ№89 по предмету "Индивидуальный проект".  
Проект включает в себя схемы, презентации, текстовое описание и исходный код.  
Проект создан в среде разработки Arduino IDE.  

Перечень функциональных возможностей прибора:  
1) Подача звонков согласно расписания (заложено 12 уроков на каждый день).  
2) подача тревожных сигналов в случае ЧС.  
3) Запуск музыкального сопровождения во время перемены.  
4) Удобный веб интерфейс для настройки расписаний и управления файлами музыкального сопровождения.
